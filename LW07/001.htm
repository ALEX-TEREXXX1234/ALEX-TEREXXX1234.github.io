Методические указания к лабораторной работе № 7

 

Подготовка исходных данных


Программе NeuroPro для обучения предоставляется информация об итогах выборов в США за 100 лет (табл. П6.1). Для всех выборов известны ответы на 12 вопросов и какая партия когда победила. После обучения нейронная сеть дает свой вариант для дополнительного набора ответов (табл. 7.2), т. е. предсказывает результаты выборов.
Данные представим в виде таблицы Exel, где в качестве полей представлены 12 вопросов, описанные выше, а в качестве записей- ответы на все эти вопросы по всем годам проведения выборов, начиная с 1860 года и заканчивая 1980. Ниже представлено описание аббревиатур всех полей:
• YEAR-год выборов;
• ANSWER- ответ (какая партия победила);
• MORE1- правящая партия была у власти более одного срока;
• MORE50- правящая партия получила больше 50% голосов избирателей на прошлых выборах;
• THIRD- в год выборов была активна третья партия;
• CONC-была серьезная конкуренция при выдвижении от правящей партии;
• PREZ- кандидат от правящей партии был президентом в год выборов;
• DEPR- год выборов был временем спада или депрессии;
• VAL2_1-рост среднего национального валового продукта на душу населения больше 2,1%;
• CHANGES- правящий президент произвел существенные изменения в политике;
• WAVE- во время правления были существенные социальные волнения;
• MIST- администрация правящей партии виновна в серьезной ошибке или скандале;
• R_HERO- кандидат правящей партии — национальный герой;
• O_HERO- кандидат оппозиционной партии — национальный герой.
Ответы на вопросы представлены в виде 0 и 1. Единица означает утвердительный ответ на вопрос, ноль — отрицательный.
Для сохранения таблицы Exel в формате dbf:
> выделите диапазон ячеек таблицы;
> присвойте диапазону имя «база_данных»;
> выполните команду Файл, Сохранить как…;
> выберите тип файла «DBF 3 (dBase III) (*.dbf)»;
> нажмите кнопку Сохранить;
> соглашайтесь со всеми задаваемыми Ехсеl вопросами.
Выходное поле обозначается как ANSWER, результатом является значение переменной, соответствующее коду выигравшей партии (1 — правящая партия, 2 — оппозиционная партия).
Аналогичным образом подготовьте статистику для работы с обученной сетью в виде табл. 1, используя данные из табл. 2 и сохраните ее в файле USA85.dbf. При этом статистика должна содержать две выборки с разными ответами в поле ANSWER (1 и 2).

Выполнение лабораторной работы


Запустите NeuroPro. Создайте новый проект:
> выполните команду Файл, Создать.
Сохраните созданный проект в файл:
>выполните команду Файл, Сохранить как...;
>в появившемся диалоговом окне Сохранение выберите нужную папку и введите имя файла, например, USA;
>нажмите на кнопку Сохранить.
В результате окно проекта будет выглядеть как на рис. 1.

Рис. 1. Окно создания проекта
Загрузите файл со статистикой задачи:
> в окне проекта нажать кнопку Открыть файл данных;
> в появившемся окне Открытие файла выбрать файл с нужной статистикой и нажать на кнопку Открыть.
В случае удачного открытия файла статистики он отображается на экране в виде таблицы (рис. 2).
Не закрывая окно с таблицей, переключитесь на окно проекта.
Примечание. Выполнение всех операций с нейросетью производится на основе статистик, при этом используется статистика открытая в данный момент. Для обучения и тестирования необходимо использовать разные статистики (с различными примерами).
Создайте новую нейросеть:
> в окне проекта нажать кнопку Новая сеть, появится окно Созлдниенейронной сети (рис. 3), в котором можно настроить параметры создаваемой нейросети.

Опишем кратко работу в окне Создание нейронной сети на вкладке Входы и выходы (рис. 3). Список Поля в файле данных позволяет выбрать конкретное поле, после чего можно настроить в правой части диалога параметры использования данного поля при создании нейросети: определить, что хранится в данном поле (поле может использоваться как один из входов нейросети, поле может использоваться как один из выходов нейросети, поле может не использоваться нейросетью), определить тип значений поля (непрерывный или дискретный), определить точность значений данного поля, которую мы хотим достигнуть при обучении нейросети (только если поле является выходным).
Первоначально при открытии окна Создание нейронной сети программа автоматически предлагает свой вариант параметров нейросети для открытой статистики, при этом входными считаются все числовые

Рис. 3. Работа на вкладке Входы и выходы
Поля кроме последнего, а выходным считается последнее числовое поле, выделенное полужирным шрифтом (рис. 3). Данные настройки можно изменять вручную.
Настройте поле YEAR (год) как поле, не используемое сетью:
> выберите в списке Поля в файле данных поле YEAR;
> включите переключатель Поле не используется сетью.
Настройте поле ANSWER (Ответ) как выходное поле сети:
> выберите в списке Поля в файле данных поле ANSWER;
> включите переключатель Поле является выходным для сети. При этом переключатель группы Тип значений поля должен быть установлен в положении Количественный (непрерывный), а в поле Точность —значение 0,1.
Настройте поле О_НЕRО (кандидат оппозиции — национальный герой) как входное поле сети:
> выберите в списке Поля в файле данных поле О_НЕRО;
> включите переключатель Поле является входным для сети. При этомпереключатель группы Тип значений поля должен быть установлен в положение Количественный (непрерывный).
В результате получим распределение входов и выходов сета как на рис. 4.
