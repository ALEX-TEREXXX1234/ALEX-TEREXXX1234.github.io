Методические указания к лабораторной работе № 8

 

Выполнение лабораторной работы


1. Запустите NeuroPro.
2. Откройте проект USA.npp, созданный в лабораторной работе № 7.
3. В окне проекта с помощью кнопки Открыть файл данных загрузите статистику USA.dbf с результатами выборов.
4.Сверните окно данных с помощью кнопки Свернуть.
5. Выполните тестирование сети командой главного меню Нейро-сеть. Тестирование. В результате на экране появится окно Тестирование USANetwork (рис. 1), в котором отображаются результаты тестирования необученной сети USANetwork.
Из результатов тестирования необученной сети видно, что сеть не дала правильный прогноз ни для одной выборки. Средняя по модулю ошибка составила примерно 0,49. Максимальная по модулю ошибка оказалась около 0,57.
6. Сохраните результаты тестирования в текстовом файле:
> выполните команду главного меню Файл, Сохранить как...;
> в появившемся окне Сохранить как введите имя файла USANetwork_test1;
> нажмите на кнопку Сохранить

Рис. 1. Результаты тестирования необученной сети USANetwork
7. Закройте приложение.

 


Отчет


Отчет по лабораторной работе должен содержать:
• наименование и цель работы;
• таблицу результатов тестирования необученной нейросети.
