Метод мозговой атаки

Метод мозговой атаки (МА) обычно используется на фазе формирования множества альтернатив. Его иногда называют методом мозгового штурма или методом коллективной генерации идей. Он применяется для самых различных целей, таких, например, как поиск вариантов технических решений, новых технологий, выбор стратегии разработки НИР и ОКР, поиск причин каких-либо явлений, построение различных перечней и т.п.
Механизм работы этого метода основан на цепной реакция ассоциаций, когда высказанная одним человеком идея рождает идеи у других участников мозговой атаки.
Рассмотрим формы, этапы и правила проведения МА.
Все участники МА делятся на две группы: в первую входит один или группа руководителей МА. (ведущих); во вторую - люди - генераторы идей (эксперты). В качестве ведущего для решения задач экспертизы, очевидно, должно выступать ЛПР, которое лаконично формулирует цель МА. и запускает механизм цепной реакции. Все высказывания экспертов должны фиксироваться либо стенографистом, либо магнитофоном. Задача экспертов заключается в генерировании ответов на поставленный вопрос. Обычно генерирование продолжается до затухания цепной реакции, что составляет от 20 мин до 2 ч.
Ведущие могут быть двух типов: активные и пассивные. Первые должны обладать искусством красноречиво направлять в русло проблемы и поддерживать процесс генерации по возможности дольше; вторые обязаны лишь следить за соблюдением правил МА. Пассивный характер ведения МА в некоторых случаях исключает ведущего из процедуры, при этом проблему оглашает один из экспертов, а высказывания производятся поочередно.
По последовательности чередования этапов и их составу различают две основные формы МА:
1) простую (по Осборну);
2) каскадную.
Рассмотрим особенности этих форм, отмеченные проф. Е.С. Жариновым. Простая МА содержит следупцие этапы и правила их выполнения:
1. Генерация: ведущий должен четко сформулировать задачи, требующие кратких ответов; запрещается критика в любой форме идей, высказанных, любым участником МА; ведущим поощряются любые короткие высказывания без аргументации, развитие предложенных идей, фантастические ассоциации и примеры.
2. Объективация: освобождение высказываний от личностных признаков путем стандартной и лаконичной записи идей; группирование идей по смыслу и назначению.
3. Селекция: выделение из предложенных идей тех, которые соответствуют задаче МА.
Для реализации второго и третьего этапов выделяют две отдельные группы экспертов, желательно из тех, которые не участвовали в экспертизе. При этом могут быть использованы самостоятельные процедуры экспертизы, например парные сравнения. Каскадная МА содержит следующие этапы и правила их реализации:
1. Поиск: реализуются все три этапа простой МА
2. Контрадиктация (противопоставление): реализуется по аналогии с предыдущим этапом, но при условии, что ту же задачу надо решить на основе идей, противоположных полученным на первом этапе; в результате выполнения первых двух этапов имеются два противоположных списка идей.
3. Синтез: совмещение двух списков идей в одной системе.
4. Прогноз: выведение на основе единого списка идей представлений о возможностях, вытекающих из решения задачи МА; критика запрещена.
5. Генерализация: обобщение всего многообразия идей и сведение их к небольшому числу принципов, из которых эти идеи могут быть выведены и без их предварительного знания; критика запрещена.
6. Деструктор (компроментация): критика полученной обобщенной оистемы идей с логической, фактической, ценностной, этической, эстетической, социальной позиций; критика участников МА запрещена.
Возможны и некоторые модификации приведенных форм МА. Участники могут быть заранее предупреждены о том, какая проблема будет обсуждаться, и, таким образом, они имеют возможность подготовиться к экспертизе. Достаточно жесткая алгоритмизация МА делает возможной ее реализацию с помощью технических средств системы управления производством и отработкой ЛА.
